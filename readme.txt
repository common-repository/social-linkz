Social Linkz plugin helps you easily share your content to social media.

== Description ==

Social Linkz is a simple, fast, and lightweight social sharing plugin.

=== Features ===

* Blazing fast. Scripts don’t run where they shouldn’t and it uses inline SVG icons without the overhead of third-party libraries.
* Easy UI with no admin menu clutter. Enable inline share buttons, floating, or both. Pick from 24+ different networks and match your site’s branding in seconds.
* Numerous social share platforms (see a list of them below)
* Many placement options before or after posts, floating over your website’s pages, via widget, via shortcode, (top right, bottom left etc.)
* The possibility to give several actions to one social share button (e.g. your Facebook icon can lead visitors to your Facebook page, and also show a Facebook button to share your page on social media)
* Share counts
* Rearrange Order of social share icons
* Lots of other customization options for your social share buttons
* Compatible with Gutenberg editor

== Installation ==

**The easy way..**

1. Go to WordPress Dashboard. Locate Plugins -> Add New
2. Search **Social Linkz** plugin using the search option
3. Find the plugin and click Install Now button
4. After installation, click on Activate Plugin link to activate the plugin.

**The hard way..**

1. Download the plugin [social-linkz.zip](https://downloads.wordpress.org/plugins/social-linkz.zip)
2. Go to WordPress Dashboard. Locate Plugins -> Add New
3. Click on the Upload Plugin link from the top
4. Upload the downloaded **social-linkz.zip** file and click on Install Now
5. After installation, click on Activate Plugin link to activate the plugin.
